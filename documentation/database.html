Database

The animals database is designed to store information about various animals, specifically focusing on dogs in this implementation. This database can be expanded in the future to include additional tables for other types of animals or related data.

Description

Database Nmeanimals

Port: 3306 (MySQL db)

Schema: The database consists of a single table, dogs, which stores information on individual dogs, such as their ID, name, breed, age, and description.

Table: dogs

Table Description: The dogs table holds details about individual dogs, including each dog's unique ID, name, breed, age, and a brief description. This information can be used in applications like pet adoption platforms, veterinary databases, or animal shelters.

Table Overview:

Column Data Type Description
dogID INT Unique identifier for each dog.
name VARCHAR(20) The name of the dog.
breed VARCHAR(40) The breed of the dog.
age INT The age of the dog in years.
description VARCHAR(400) A detailed description of the dog, especially personality.

Constraints: dogID: not null, primary key, auto increment

Data Example:

dogID name breed age description
1 Rex German Shepherd 4 Suitable as a house sitter, does not get along with other animals.
2 Lisa American Pit Bull Terrier 3 Very friendly dog also suitable for children.
