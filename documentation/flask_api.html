Flask Rest API

In this page you can find an overview of Flask Rest API

Functions

  1. get_dogs_db()

    Task: Retrieves all dog records from the database and returns them as JSON.
    Arguments: None
    Output: Database results as JSON
    Database Interaction: Connects to MySQL, fetches data, then closes the connection.
    Error Handling: If a database error occurs, it returns a JSON error response with a 500 status code.

Endpoints

  1. /test_db

    Method: GET
    Description: A testing endpoint that returns the list of dogs using the get_dogs_db() function.
    Response: JSON object containing dog data or an error message (500 status code).

  2. /api/dogs

    Method: GET
    Description: Retrieves all dog records from the database and returns them in JSON format.
    Response: JSON object containing dog data or an error message with (500 status code).

  3. /api/adoption

    Method: POST
    Description: Adds a new dog adoption record to the database.
    Request Body:

    • dogName: Name of the dog.
    • breed: Breed of the dog.
    • age: Age of the dog.
    • description: Description of the dog.
    Response: Returns a success message with a 200 status code or an error message with a 500 status code.

  4. /health

    Method: GET
    Description: An endpoint for checking the health of the application.
    Response: JSON object indicating application status as "healthy" with a 200 status code.
