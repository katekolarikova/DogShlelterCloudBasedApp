Project Overview

This documentation provides information about the architecture and implementation of a web application designed for dog adoption and fostering. The important part of this implementation is using Docker and Kubernetes to efficiently manage and deploy the application and enhance its scalability, security, and workflow efficiency. This project was implemented as a part of course DAT515 Cloud Computing.

Our application consists of three primary components: a MySQL database, a Flask backend, and a React frontend served through Nginx. The MySQL database is configured with a persistent volume to ensure data durability. The Flask framework takes care of handling data between the frontend and database by providing a channel in the form of a REST API. The frontend, developed in React, provides an interface for users to browse available dogs for adoption, as well as submit new dog adoption requests through a web form.

By utilizing Docker containers, we encapsulate each component of our application, ensuring consistency across different development and production environments. Kubernetes manages these containers and adds important features such as scalability or enhancing the security.
