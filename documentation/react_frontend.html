React Frontend

The React frontend provides an interactive user interface for viewing and adding dog adoptions. Users can browse available dogs or submit a form to offer a dog for adoption.

Technologies

  1. React: core framework for building the UI.
  2. Bootstrap: library to provide styling and responsive layout.

Components

  1. App: The main application component that handles routing and renders the navigation bar that includes the links to the homepage and adoption form.
  2. Homepage: Fetches a list of dogs currently available for adoption from the API and uses a DogCard component to display each dog’s details.
    • DogCard: A reusable component that renders a dog’s name, breed, age, and description within a Bootstrap card layout.
  3. Form: A submission form for dog adoptions, collecting both dog and owner details and submitting them to the backend API.

Routing

  1. React Router: Used to manage application routes:
    • /: Displays the Homepage component with a list of available dogs.
    • /give_for_adoption: Displays the Form component, allowing users to submit an adoption form.

API Integration

  1. GET /api/dogs (Homepage):
    • Used in the Homepage to display all dog records.
    • Handles errors in fetching data and displays an error message if loading fails.
  2. POST /api/adoption (Form):
    • Used in Form to submit the adoption form data to the backend API.
    • Provides feedback to the user based on the response, indicating success or failure.
